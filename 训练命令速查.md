## MOSI 主模型 (3-seed, 需P6K checkpoint)

```bash
CKPT="outputs/P6K/text_audio_conservative_s42_s42_20260619_031645/best_model.pth"

python scripts/train_textft_lora_mainline.py \
```

## MOSI 消融 (5变体×3seeds, 需P6K checkpoint)

```bash
CKPT="outputs/P6K/text_audio_conservative_s42_s42_20260619_031645/best_model.pth"
BASE="configs/experiments/p6ak_mosi_final_ablation"

# A1: 纯文本 (最快, ~3 min)
python scripts/train_textft_lora_mainline.py \
  --config $BASE/A1_text_only_p6k_init_s42.yaml \
  --init_checkpoint $CKPT

# A2: 无音频修正 (~7 min)
python scripts/train_textft_lora_mainline.py \
  --config $BASE/A2_no_audio_correction_s42.yaml \
  --init_checkpoint $CKPT

# A3: 无视觉修正 (~7 min)
python scripts/train_textft_lora_mainline.py \
  --config $BASE/A3_no_vision_correction_s42.yaml \
  --init_checkpoint $CKPT

# A4: 无门控 (~7 min)
python scripts/train_textft_lora_mainline.py \
  --config $BASE/A4_no_reliability_gate_s42.yaml \
  --init_checkpoint $CKPT

# A5: 无交互 (~7 min)
python scripts/train_textft_lora_mainline.py \
  --config $BASE/A5_no_interaction_s42.yaml \
  --init_checkpoint $CKPT

# 替换 s42 → s2024, s3407 运行其余seed
```
- 全部15个约1.5h

## 常用参数

| 参数 | 说明 |
|------|------|
| `--config` | YAML配置文件路径 (必需) |
| `--init_checkpoint` | P6K权重初始化路径 (MOSI必需) |
| `--smoke` | 1 epoch + 5 batch 快速测试 |
| `--device` | 设备 (默认cuda) |
| `--max_epochs` | 覆盖config中的epoch数 |
