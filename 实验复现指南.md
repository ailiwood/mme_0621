## 六、复现关键注意事项

1. **P6K checkpoint 是必需的**（MOSI 主模型）。无此文件时，仅能复现 P6AG T1（约 82%）。
2. **所有数据路径在 config YAML 中指定**，需确认 `feature_root` 和 `csv_path` 指向正确位置。
3. **每个 seed 的 test 仅评估一次**。重新训练会覆盖 `outputs/` 中的旧结果。
4. **MOSEI strict cohort (18,571) 尚未训练**。当前 MOSEI 结果使用 20,680 样本版本。
5. **GPU 显存需求**：batch_size=8 + grad_accum=8 使用约 4-6GB 显存。如显存不足，增大 grad_accum_steps 并减小 batch_size。
