## configs/ — 实验配置

```
configs/
├── canonical/mosei/          ← MOSEI 最终 AWAF configs
│   ├── control_awaf_slstm_s42.yaml     # F0 主模型 (12-epoch)
│   ├── awaf_no_interaction_s42.yaml    # F5 无交互消融
│   ├── fusion_mean_s42.yaml            # F4 固定等权
│   ├── fusion_gated_s42.yaml           # 门控变体
│   └── encoder_lstm_s42.yaml           # E1 LSTM
├── baselines/                ← Baseline-lite configs
│   ├── mosei/                # MOSEI baselines (T+A+T+V)
│   └── mosi/                 # MOSI baselines (未训练)
├── experiments/              ← MOSI 最终实验 configs
│   ├── p6aj_mosi_87_recovery/          # 主模型 3-seed
│   └── p6ak_mosi_final_ablation/       # 消融 5×3=15 configs
└── references/               ← 历史参考 configs
    └── mosi_conservative/              # P6K T+A conservative
```

```
models/
├── textft_lora_xlstm_awaf_residual.py  ← 主模型类 (所有mode入口)

│   ├── awaf.py                          ← MOSEI AWAF融合 (3阶段)

│   └── slstm.py                         ← sLSTM时序编码器

├── modules/

│   ├── uncertainty_residual_gate.py     ← 残差门控(P6K参考)
│   └── text_confidence_residual.py      ← 文本置信度残差
├── interaction/
│   └── cross_modal_transformer.py       ← 跨模态Transformer
└── baselines/                           ← 7个baseline实现
    ├── base_baseline.py
    ├── tfn_lite.py / lmf_lite.py / mult_lite.py
    ├── self_mm_lite.py / misa_lite.py
```
